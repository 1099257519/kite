项目初始化成功

您的网站已经初始化完成

现在请点击下方按钮关闭cli(或者手动 ctrl + c 强制退出 <_<)

可以进行两种操作

pro1.生产环境

pro1.1 在cmd 中输入 npm run server 即可进入程序

pro1.2 (url或者ip)+ :8086端口即可看到客户端主页

pro1.3 (url或者ip)+ :8086/admin端口即可看到客户端后台页面 admin(是默认的，就是在上一步没有填写的时候)

dev1.本地开发环境

dev1.1 在cmd 中输入 npm run server-dev 即可进开启接口服务

dev1.2 在cmd 中输入 npm run admin-dev 即可进入后台开发预览（地址为：localhost:8083）

dev1.3 在cmd 中输入 npm run client-dev 即可进入前台开发预览（地址为：localhost:8081）

dev1.4 开发环境下 一定要先运行dev1.1的情况下再运行 dev1.2 或者 dev1.3

dev1.5 打包后台界面 npm run admin-build

dev1.6 打包前台界面 npm run client-build
